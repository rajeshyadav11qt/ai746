Connections:
GPS		Rasp
VCC	=	Pin 2
GND	=	Pin 6
TXD	=	Pin 10
RXD	=	No need

(TXD is for Data o/p)
(RXD is for Data i/p)



Commands & Code:

sudo nano/boots/config.txt

(write the following in the txt file)

dtparam=spi=on
dtoverlay=pi3-disable-bt
core freq=250
enable_uart=1
force_turbo-1

(now to close the txt file ctrl+O -> Enter -> ctrl+x)

(Reboot Raspberry Pi using:)
sudo reboot

(once rebooted do following commands)

sudo systemctl stop serial-getty@ttyS0.service
sudo systemctl disable serial-getty@ttyS0.service

(Reboot Raspberry Pi using:)
sudo reboot

(once rebooted do following commands)

sudo systemctl enable serial-getty@ttyAMA0.service

sudo apt-get install minicon

sudo cat/dev/ttyAMA0
